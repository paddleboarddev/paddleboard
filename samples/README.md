# Samples

Self-contained sample projects for demoing and exercising PaddleBoard.

| Sample | What it is |
|---|---|
| [`demo/`](demo/) | **Paddle Log** — a small FastAPI service plus a presenter script that walks through every headline PaddleBoard feature in order. |

Each sample is standalone: open it as its own PaddleBoard window so the file
tree, git state, and agent threads are scoped to it.
